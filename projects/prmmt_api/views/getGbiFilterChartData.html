This method retrieves table data via GlideAjax, filters the data via query parameters and populates it in a HighCharts visualization tool such as a bar or pie chart. This function come with two parameters. The "filter" parameter returns the column name in which the data will be filtered and the "callback" parameter returns the HighCharts visualization tool.

There are two components you must have to ensure the tool will render on your page. A third is optional depending on the filter type.

  1. Visualization Component - html container where the tool displays
  2. Script Component- ensures the rendering of the tool
  3. Filter Component(Optional) - html filter container with query param update function
